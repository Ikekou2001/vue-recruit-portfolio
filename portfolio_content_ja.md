## RUN_2: プロフィール画像の推奨事項

### プロフェッショナルな写真を使用する場合

#### ✅ 推奨される写真
- **背景**: 
  - 白や明るいグレーなどシンプルな単色背景
  - カフェやオフィスのぼやけた背景（適度にプロフェッショナル）
  - 自然光の入る明るい場所で撮影したもの
  
- **服装**: 
  - ビジネスカジュアル（襟付きシャツ、ポロシャツなど）
  - 清潔感のある服装
  - IT企業向けなら、過度にフォーマルすぎなくてOK
  
- **表情・構図**: 
  - 自然な笑顔または穏やかな表情
  - 正面または斜め45度から撮影
  - 胸から上が写るバストショット
  - カメラ目線で自信を持った印象を与える

#### ❌ 避けるべき写真
- プライベート感の強い写真（旅行、飲み会など）
- 暗すぎる・ぼやけている写真
- 集合写真や複数人が写っている写真
- 過度に加工された写真
- カジュアルすぎる服装（Tシャツ、パーカーなど）

### 写真がない場合の代替案

#### オプション1: イニシャルデザイン
```
背景色: ネイビーブルー (#2C3E50) または深緑 (#27AE60)
文字: 白色で "K.I" または "KI"
フォント: モダンで読みやすいサンセリフ体（Inter, Roboto, Noto Sans JPなど）
形状: 円形または角丸正方形
```

#### オプション2: シンプルなアイコン
```
- GitHubのデフォルトアバターを洗練されたものにカスタマイズ
- 幾何学的なパターン（Identicon風）
- プログラミングやAIを連想させるミニマルなアイコン
  例: 括弧{}、コードブロック、ニューラルネットワークのシンプルな図
```

#### オプション3: AI生成アバター
```
- プロフェッショナルな雰囲気のAIアバター
- 実写風ではなく、イラスト調やベクター調
- カラーパレット: 落ち着いたトーン（青、緑、グレー系）
```

### Web用プロフィール画像のベストプラクティス

1. **サイズと解像度**
   - 推奨サイズ: 400×400px 〜 800×800px
   - 形式: PNG（透過対応）またはJPG
   - ファイルサイズ: 200KB以下（ページ読み込み速度のため）

2. **アスペクト比**
   - 1:1（正方形）が必須
   - 円形表示にも対応できるよう、顔は中央に配置

3. **色とコントラスト**
   - サイト全体のカラースキームと調和させる
   - 背景と被写体のコントラストを十分に確保
   - ダークモード・ライトモード両方で見やすい画像

4. **複数デバイス対応**
   - スマートフォンでも認識しやすい明瞭さ
   - Retinaディスプレイ対応（2x解像度も準備）

### 推奨ツール
- **写真撮影**: スマートフォンのポートレートモード、自然光
- **背景除去**: remove.bg（https://www.remove.bg/ja）
- **画像編集**: Canva（https://www.canva.com/）、Figma（無料）
- **イニシャル作成**: UI Avatars（https://ui-avatars.com/）
- **AI生成**: Microsoft Designer、Leonardo.ai
